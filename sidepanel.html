How to use Y/TXT

Basic workflow
Select text on any page → open the extension → pick a tool. Your selection is pre-filled automatically. Hit Run Tool and the result appears below.
Chained tools
A tool can now define multiple sequential steps. Each step receives the previous step's output as {{input}} and {{previous}}, while the first step also has the original source available as {{originalInput}}.
+ selection / = selection
Select text before opening the popup — it fills the input automatically. Already inside a tool? Select more text elsewhere, reopen the popup and use + selection to append or = selection to replace. Your workspace is preserved across opens.
Staged Content
Extra context sent alongside your input. Useful for code reviews where you want to provide the full file while only pasting a snippet in the main input.
Recent
Stores the last 50 runs. Click any entry to reopen that tool with the original input and result pre-filled. Use it to iterate on a previous run or compare results.
Providers & API keys
Go to ⚙ Options to set your provider and API key. Nvidia Build offers a free tier — good for everyday tasks. Keys are stored locally in your browser only.
Custom tools
Edit the Tools JSON in Options to add your own. Use {{input}} for the active step input, {{previous}} inside chains, and option IDs like {{targetLanguage}}. Add a steps array to build chained tools. Changes take effect on next popup open.
Custom models
Edit the Models JSON in Options to map tier names (default, fast, coding, creative) to any model your provider supports. Changes take effect immediately.
